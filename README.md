# IPL-DATA-VISUALISATION--EXCEL
An interactive Excel dashboard project analyzing IPL historical data to derive insights on team performance,player statistics,match trends and key performance indicators using Pivot Tables,Power Query and dynamic charts.
📌 Overview

This project is an interactive IPL Dashboard created using Microsoft Excel.
The dashboard analyzes match data from the Indian Premier League and provides clear insights into team performance, player statistics, and match trends.

The analysis is based on data from the Indian Premier League.

🎯 Objective

Analyze IPL match data

Compare team performances

Create an interactive dashboard using Excel

📊 Features

Season-wise analysis

Team performance summary

Top batsmen and bowlers

Toss decision analysis

Venue analysis

Interactive slicers and filters

🛠 Tools Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Conditional Formatting

Excel Formulas (SUMIFS, COUNTIFS, IF, XLOOKUP)

📂 Dataset

The dataset includes:

Match details (Season, Date, Venue)

Team names

Toss winner and decision

Match winner

Player performance statistics
